# How To Use AGENTS.md

A global `AGENTS.md` file is not a valid place to store information that is specific to a single task.

Information may be stored in a global `AGENTS.md` file only when it covers at least 80% of use cases.

If guidance applies only to a narrow task, tool, or situation, it should not be stored in a global `AGENTS.md` file.

Do not put narrow task-specific guidance into `AGENTS.md`.
