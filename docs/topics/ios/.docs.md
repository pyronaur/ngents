This topic is organized around `docs`, `skills`, `hig-doctor`, and `app-store-connect-cli-skills`.

- Prioritize `hig-doctor` first when you need Apple HIG guidance
- Use `docs topic ios docs` when you need iOS reference docs like `ng-hig-doctor.md` and `SOSUMI.md`.
- Use `docs topic ios skills` when you need an on-demand iOS, Swift, SwiftUI, SwiftData, or Apple-tooling skill.
- Use `docs topic ios hig-doctor` when you need curated Apple HIG skills and references.
- Use `docs topic ios app-store-connect-cli-skills` when you need App Store Connect CLI automation workflows for builds, TestFlight, metadata, signing, submissions, or screenshots.
- Prefer `hig-doctor` before raw Apple docs when both could answer the question.
